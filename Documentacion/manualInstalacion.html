Instalación de tema en WordPress.

Cargar el tema usando el panel de administración de WordPress.
Ingresando desde el url ej. http//dominio.com/wp-admin.
Desde el panel izquierdo en Apariencia > temas.
Click en ‘añadir nuevo’.
Click en ‘subir tema’.
Cargar el tema archivo llamado -good-food.zip.



Una vez que el tema este cargado se debe activar.



Al ser activado WordPress te recordara que deben ser descargados y activados los siguientes plugins.

CMSMasters Content Composer
CMSMasters Contact Form Builder
CMSMasters Mega Menu
CMSMasters Importer
CMSMasters Custom Fonts
Revolution Slider
Layer Slider

Ajustes de Apariencia del tema.

1. Esquemas de color.

Los esquemas de color le permiten crear soluciones de color preestablecidas que posteriormente pueden aplicarse al editar el contenido de su sitio web, como menús, varios códigos cortos, etc.
Las combinaciones de colores se pueden configurar y editar en Configuración del tema - pestaña Colores en el panel de administración.
Inicialmente, el tema viene con los esquemas (pero puede agregar más, lo veremos más adelante en esta sección), que son:

Predeterminado: se aplica a la mayoría del contenido del cuerpo, a menos que se especifique otro.

Encabezado: se aplica a Encabezado, pero también se puede usar para otras secciones del sitio web.

Navegación: se aplica a la navegación, pero también se puede usar para otras secciones del sitio web.

Header Top: se aplica a Header Top Line, pero también se puede usar para otras secciones del sitio web.

Pie de página: se aplica al pie de página, pero también se puede usar para otras secciones del sitio web.

Personalizado 1: se utiliza para varios elementos a su elección.

Personalizado 1: se utiliza para varios elementos a su elección.

Personalizado 3: se utiliza para varios elementos a su elección.


2. Configuración de fuente.

Todas las configuraciones de fuente del sitio web se pueden editar en Configuración de tema - pestaña Fuentes. Puede encontrar configuraciones para contenido general, enlaces, navegación, encabezados y algunos otros elementos. Para personalizar la apariencia de su sitio web, simplemente juegue con la configuración de fuente, o deje el tema por defecto si lo desea.


3. Encabezado, pie de página, logotipo y Favicon

En la última versión de WordPress para agregar un Favicon, debe proceder a Apariencia - Personalizar y hacer clic en la Identidad del sitio.
En Configuración de tema - General - pestaña General puede administrar algunos logotipos.
En Configuración de tema - General - pestaña Encabezado puede administrar algunas configuraciones de encabezado, como contenido HTML personalizado en su encabezado, contenido de primera línea, etc.
Tenga en cuenta: para un rendimiento superior en las pantallas Retina, la imagen del logotipo se agrega en dos tamaños. Se recomienda que agregue una imagen para el logotipo de Retina dos veces más grande que su logotipo predeterminado.

Traduce tu página.

No hay archivos de idioma listos, pero solo necesita seguir unos pocos pasos para traducir su sitio web al idioma necesario. Puedes traducir el tema a cualquier idioma, así como a RTL Language.
Está listo para RTL, por lo que puede agregar un complemento y cambiar el contenido del tema para que se muestre como RTL.
El primer paso que debe tomar es crear su propio archivo de idioma. Para implementar esto debes instalar Poedit: www.poedit.net
Copie el archivo good-food.pot a su computadora.
Nota: Encontrará el archivo pot de idioma en wp-content / themes / good-food / theme-vars / languages.
Ejecute Poedit y proceda a Archivo / Nuevo catálogo desde archivo POT.



Elija el archivo .pot que ha copiado para usar como un esqueleto de su idioma. Obtendrá la lista de mensajes y frases comunes.
Seleccione el texto que desea traducir a su idioma e ingrese los cambios en un campo inferior como se muestra en la captura de pantalla:



Debe ingresar texto nuevo al campo en la parte inferior de la ventana, se guardará para cada elemento automáticamente. Edite todos los mensajes que necesita y asegúrese de guardar los cambios en el archivo antes de cerrar.
Cree un archivo de idioma (por ejemplo, el archivo lg_LG o cualquier otro nombre que prefiera). El programa creará dos archivos con nombres similares con extensiones .po y .mo.
Cuando haya terminado y guardado la traducción, copie estos dos archivos en su servidor: la carpeta wp-content / languages / themes denominada good-food-lg_LG.po y good-food-lg_LG.mo. En caso de que no haya una carpeta wp-content / languages / themes, primero debe crearla.
Aquí hay una lista de ejemplos de códigos de idiomas que se pueden usar:
de_DE para la traducción alemana
fr_FR para la traducción francesa
it_IT para la traducción italiana
pt_PT para la traducción al portugués
es_ES para la traducción española
ru_RU para la traducción rusa
etc.
Puede usar letras o números para nombrar su idioma, solo asegúrese de agregar el mismo nombre al archivo wp-config.
